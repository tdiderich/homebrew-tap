# tdiderich's Homebrew tap

Homebrew formulae for tools I maintain.

## Usage

```bash
brew tap tdiderich/tap
brew install <formula>
```

Or install directly without tapping:

```bash
brew install tdiderich/tap/<formula>
```

## Formulae

### kazam

Beautiful static sites from simple YAML. One Rust binary, no framework, no npm, no runtime JS. Designed so AI agents can author your site end-to-end.

```bash
brew install tdiderich/tap/kazam
```

- Source: https://github.com/tdiderich/kazam
- Docs: https://tdiderich.github.io/kazam/
